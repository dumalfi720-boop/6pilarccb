MODULE 5.7

💙 Where to Find Ready Skills

Don't reinvent the wheel. The Skills ecosystem already has repositories, marketplaces and community curation with hundreds of skills tested and ready to use. Know where to look.

TOPIC 1

⭐ awesome-claude-code

The repositoryawesome-claude-codeon GitHub and the most complete curated list of resources for Claude Code. Includes exemplary skills, configurations, CLAUDE.md and tips from the community.

🎯Main Concept

  • •URL: github.com/anthropics/awesome-claude-code
  • •Content:Skills, CLAUDE.md templates, tips, workflows, integrations
  • •Maintenance:Official from Anthropic, updated frequently by the community

💡Practical Tip

Star the repo and turn on notifications. New skills are added weekly by the community via Pull Request. It's the best place to discover patterns and techniques you didn't know existed.

TOPIC 2
2

🤖 awesome-agent-skills

The repositoryawesome-agent-skillsfocuses specifically on skills for AI agents, including but not limited to Claude Code. Skills here tend to be more generic and portable across different platforms.

🎯Main Concept

  • •Focus:Platform-agnostic skills, but compatible with Claude Code
  • •Categories:Development, research, writing, data analysis, automation
  • •Quality:Variable - each skill has a rating and reviews from the community

💡Practical Tip

Skills from this repo may need adaptation to work in Claude Code. Check that they use $ARGUMENTS correctly and that the steps are compatible with the available tools (Read, Write, Bash, etc.). Adapt whatever you need.

TOPIC 3
3

⚡ Superpowers

Superpowersis a curated collection of high-performance skills for Claude Code, focused on productivity and automation of complex workflows.

🎯Main Concept

  • •Differentiators:Skills optimized for minimal token consumption
  • •Categories:Code gen, refactoring, testing, documentation, DevOps
  • •Format:Each skill comes with README, usage examples and expected output

💡Practical Tip

The difference between Superpowers is token optimization. If you are on the plan with usage limits, prioritize skills from here - they were designed to maximize results with minimum consumption. Each token saved and budget for more tasks.

TOPIC 4
4

🏪 SkillsMP (Skills Marketplace)

O SkillsMPand a dedicated marketplace where creators publish and share skills with the community. It works as an "app store" for Claude Code.

🎯Main Concept

  • •Search:Search by category, language, rating or use case
  • •Installation:One-click install that copies the .md to your commands folder
  • •Reviews:Rating system and community comments
  • •Publication:Anyone can publish skills after basic verification

💡Practical Tip

Filter by skills with more than 4 stars and more than 10 reviews. Popular and highly rated skills have passed community testing. New skills without reviews may work, but you are the beta tester. Read the code before installing anything.

TOPIC 5
5

🏠 SkillHub

SkillHubIt is an aggregator that compiles skills from multiple sources into a single searchable place. Useful when you don't know which source to look for.

🎯Main Concept

  • •Aggregation:Pulls skills from awesome-claude-code, SkillsMP, Superpowers and independent repos
  • •Unified search:A single search bar for all fonts
  • •Comparison:When there are similar skills, show them side by side

💡Practical Tip

Use SkillHub as a starting point when you don't know what you're looking for. Search by problem ("code review security") instead of solution ("OWASP scanner skill"). Semantic search works best with problem descriptions.

TOPIC 6
6

🌐 awesomeclaude.ai

The websiteawesomeclaude.aiand a web directory with a visual interface to explore the entire Claude ecosystem: skills, MCPs, plugins, templates and tutorials.

🎯Main Concept

  • •Interface:Visual web with preview of each skill before installing
  • •Categories:Development, Business, Creative, Research, DevOps
  • •Curation:Editorial team that selects and highlights the best skills
  • •Tutorials:Step-by-step guides to creating and publishing your own skills

💡Practical Tip

The "Featured" section of awesomeclaude.ai is editorially curated and changes weekly. It's the best place to discover innovative skills you didn't know you needed. Set aside 10 minutes a week to explore the highlights.
